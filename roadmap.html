  • Indicateur 💬 : pastille dorée sur les posts qui ont reçu des commentaires

V1.8 — Trace GPS, AIS & édition ⏳ En cours

L'objectif : avoir une trace propre en mer sans avoir à laisser le téléphone allumé pendant des jours. Plusieurs sources se complètent et s'auto-nettoient.

Principe directeur — Pour une longue traversée, garder un iPhone écran-allumé est irréaliste. La V1.8 introduit donc plusieurs sources GPS qui se relaient :
→ AIS (transpondeur du bateau, capté par satellite) pour le suivi continu en mer
→ Import GPX du calculateur de bord (B&G / Raymarine / Garmin) pour la haute fidélité
→ iPhone GPS avec verrouillage écran pour les sorties courtes (annexe, marina, à terre)
→ Marqueurs manuels 1 tap pour les évènements ("on empanne", "dauphins")

Vie privée & sécurité — la sync AIS peut être mise en pause à volonté (zones à risque, transits sensibles, contraintes d'assurance), et la diffusion sur la page publique est différée par défaut de 48 h, ou contrôlée manuellement par le skipper. La famille connectée voit toujours en temps réel.

V3.9 — Suivi AIS multi-bateaux ⭐ 💤 Si besoin

Étendre le pipeline AIS à plusieurs bateaux simultanément — bateaux copains sur le même parcours, flotille famille-amis lors d'étapes communes, rallye ARC / OCC. Chaque bateau a sa trace colorée sur la carte et sa propre ligne dans les Réglages AIS.

  • Table boats en base : chaque bateau a son MMSI, sa clé AIS, sa couleur, son indicatif. Le bateau actuel (Wang'ap) devient le "primaire".
  • Worker Fly.io modifié : un seul WebSocket par clé AIS (capable de filtrer plusieurs MMSI), routage des frames vers le bon bateau côté serveur.
  • UI Réglages → 📡 AIS devient une liste avec bouton "+ Ajouter un bateau". Chaque ligne : nom + MMSI + couleur + statut + Modifier / Supprimer.
  • Carte : légende en bas-à-droite avec une case à cocher par bateau. Traces côte-à-côte avec opacité réduite pour les passages parallèles.
  • Cas d'usage : suivi d'un bateau copain pendant la traversée du Pacifique, flottille familiale aux Antilles, opérateur de charter avec plusieurs bateaux loués.
  • Coût : la même VM Fly.io supporte une vingtaine de bateaux sans surcoût ; les limites de l'AISStream gratuit s'appliquent par clé API, pas par MMSI.
